Справка: Список клиентов DHCP

В этом окне содержится информация по Имени клиента, MAC-адресу, Назначенному IP-адресу и Сроку действия адреса для каждого DHCP-клиента, подключенного к маршрутизатору.

  • Имя клиента - Имя DHCP-клиента.
  • MAC-адрес - MAC-адрес DHCP-клиента.
  • Назначенный IP-адрес - Присвоенный маршрутизатором IP-адрес DHCP-клиента.
  • Срок действия адреса - Срок действия адреса DHCP-клиента.

Указанные выше значения не могут быть изменены в этом окне. Чтобы обновить страницу и получить информацию по подключенным на данный момент устройствам кликните по кнопке Обновить.
