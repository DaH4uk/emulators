Справка: Фильтрация МАС-адресов

Функция Беспроводной фильтрации по MAC-адресам позволяет вам контролировать беспроводные станции, обращающиеся к Точке доступа, благодаря определению MAC-адреса обращающейся станции.

  • MAC-адрес - MAC-адрес беспроводной станции, которая хочет получить доступ.
  • Описание - Простое описание данной беспроводной станции.
  • Состояние - Состояние данной записи, либо Включена, либо Отключена.

Для отключения функции Беспроводной фильтрации по MAC-адресам оставьте параметр по умолчанию Отключена без изменений.

Для создания записи нажмите Включить, и следуйте приведённым ниже инструкциям:
Во-первых, вы должны определиться, будут ли незарегистрированные беспроводные станции иметь доступ к Точке доступа. Если вы хотите, чтобы незарегистрированные беспроводные станции имели доступ к Точке доступа, тогда вам следует выбрать радиокнопку Запретить доступ станциям, указанным во включенных записях, в противном случае следует выбрать радиокнопку Разрешить доступ станциям, указанным во включенных записях.

Для добавления записи Беспроводной Фильтрации по MAC-адресам, нажмите кнопку Добавить... и следуйте представленным ниже инструкциям:

  1. Введите соответствующий MAC-адрес в поле MAC-адрес. Формат MAC-адреса следующий: XX-XX-XX-XX-XX-XX (где X представляет собой любое шестнадцтеричное число). Например, 00-0A-EB-B0-00-0B.
  2. Введите простое описание беспроводной станции в поле Описание. Например, Беспроводная станция A.
  3. Состояние - Ввыпадающем списке Состояние для данной записи выберите Включена или Отключена.
  4. Нажмите кнопку Сохранить для сохранения данной записи.

для добавления следующей записи повторите шаги 1~4.

Для изменения или удаления существующей записи:

  1. Нажмите кнопку Изменить или Удалить в колонке Изменить в таблице Фильтрации по MAC-адресам.
  2. Введите желаемые значения на странице Добавление или Внесение Изменений в Запись Беспроводной Фильтрации по MAC-адресам, и нажмите кнопку Сохранить.

Вы можете нажать кнопку Включить все для включения всех записей, нажать кнопку Отключить все для отключения всех записей, нажать кнопку Удалить все для удаления всех записей.

Нажмите кнопку Следующая для перехода на следующую страницу или нажмите кнопку Предыдущая для возвращения на предыдущую страницу.

Внимание: Если вы включаете эту функцию и выбираете Разрешить доступ станциям, указанным во включенных записях для Правил фильтрации, и в списке нет активированных записей, тогда ни одна беспроводная станция не сможет выйти на точку доступа.
