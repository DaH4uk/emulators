Справка: Настройка цели

Маршрутизатор, располагая удобной и эффективной функцией контроля доступа, позволяет контролировать доступ узлов локальной сети к Интернет-ресурсам. В этом окне мы можете создавать список целей для конкретного узла, в котором определяется наличие или отстутсвие доступа.

  • Имя цели - Содержит имя цели, которое является уникальным.
  • Информация - В качестве цели может быть указан IP-адрес, порт или имя домена.
  • Изменить - Изменение или удаление существующей записи.

Например: вы хотите ограничить доступ для узла локальной сети с MAC-адресом 00-11-22-33-44-AA только к сайту www.google.com, Для этого необходимо выполнить следующие действия:

  1. Кликните по кнопке "Добавить...", чтобы перейти на страницу "Настройки списка цели".
  2. В поле "Режим" выберите из выпадающего списка значение "Доменное имя".
  3. В поле "Имя цели" укажите уникальное имя цели. (например, Target_1)
  4. В поле "Имя домена" введите www.google.com.
  5. Нажмите Сохранить чтобы сохранить сделанные изменения.

После этого Вы вернетесь обратно в окно "Цель" и увидите следующее:

ID Имя цели Информация Изменить
1 Target_1 www.google.com Изменить/Удалить

Нажав кнопку Добавить..., вы создадите новую запись.

Нажав кнопку Удалить все, вы очистите содержимое табицы.

Нажав кнопку Следующая, вы перейдете в следующее окно, а нажав кнопку Предыдущая, вы вернетесь на предыдущую страницу.
