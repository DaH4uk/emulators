Справка: Статический IP-адрес

Параметры IP предоставляются вашим поставщиком Интернет-услуг.

  • IP-адрес - Это IP-адрес WAN, который видят внешние пользователи через сеть Интернет (включая вашего поставщика Интернет-услуг). Введите IP-адрес в поле.
  • Маска подсети - Маска подсети используется для IP-адреса WAN. Как правило она имеет вид 255.255.255.0.
  • Основной шлюз - При необходимости введите Основной шлюз в пустое поле.
  • Предпочитаемый DNS-сервер - Введите IP-адрес DNS в пустое поле.
  • Альтернативный DNS-сервер - В случае, если ваш поставщик Интернет-услуг предоставляет другой IP-адрес DNS, введите его в данном поле.

Нажмите кнопку Далее для продолжения или кнопку Назад для возврата на предыдущую страницу.
