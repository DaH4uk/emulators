Справка: Настройки узла

Маршрутизатор, располагая удобной и эффективной функцией контроля доступа, позволяет контролировать доступ узлов локальной сети к Интернет-ресурсам. В этом окне Вы сможете создать список узлов, необходимый для Правила контроля доступа.

  • Имя узла - Здесь необходимо указать имя узла, которое должно быть уникальным.
  • Информация - В этом поле отображается информация по узлу. Это может быть IP- или MAC-адрес.
  • Изменить - Изменить или удалить существующую запись.

Например: Вы хотите ограничить доступ к Интернет-ресурсам для узла с MAC-адресом 00-11-22-33-44-AA. Для этого Вам сначала необходимо выполнить следующие действия:

  1. Кликните по кнопке Добавить..., чтобы перейти на страницу "Настройки списка узлов".
  2. В поле "Режим" выберите из выпадающего списка значение "MAC-адрес".
  3. В поле "Имя узла" укажите уникальное имя узла. (Например, Host_1)
  4. В поле "MAC-адрес" введите 00-11-22-33-44-AA.
  5. Нажмите кнопку Сохранить, чтобы сохранить внесенные изменения.

После этого Вы вернетесь в окно "Настройки узла" и увидите следующее:

ID Имя узла Информация Изменить
1 Host_1 MAC: 00-11-22-33-44-AA Изменить/Удалить

Чтобы создать новую запись по узлу, нажмите кнопку Добавить....

Чтобы очистить таблицу, нажмите кнопку Удалить все.

Нажав кнопку Следующая, вы перейдете в следующее окно, а нажав кнопку Предыдущая, вернетесь в предыдущее.
