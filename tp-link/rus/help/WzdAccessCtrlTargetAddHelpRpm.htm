Справка: Создать или изменить запись для цели

В этом окне вы можете создать запись в списке целей.

  • Имя цели - В этом поле укажите имя цели. Имейте в виду, что данное значение должно быть уникальным.
  • Режим - Здесь нужно из выпадающего списка выбрать одно из двух значений - "IP-адрес" или "Доменное имя".

При выборе IP-адрес Вы увидите следующие поля:

  • IP-адрес - Укажите IP-адрес (или диапазон адресов) цели (целей) - например, 192.168.0.23.
  • Порт назначения - Укажите порт или диапазон портов. Для портов общих сервисов можете воспользоваться полем "Стандартный порт сервиса", расположенным ниже.
  • Протокол - Здесь из выпадающего меню необходимо выбрать один из четырех вариантов - All, TCP, UDP и ICMP.
  • Стандартный порт сервиса - Здесь содержатся список портов обычных сервисов. Необходимо выбрать один из выпадающего меню, при этом номер соответствующего порта появится в поле "Порт назначения". Например, если Вы выбрали "FTP", то в поле "Порт назначения" появится значение "21".

Если было выбрано Доменное имя, то вы увидите следующие поля:

  • Доменное имя - Здесь вы можете указать четыре имени, причем это может быть как полное имя, так и ключевое слово (например, google). Доменное имя, содержащее ключевые слова (www.google.com, www.google.cn) будет либо заблокировано, либо разрешено.

Нажмите кнопку Далее для продолжения или по кнопке Назад, чтобы вернуться на предыдущую страницу.
