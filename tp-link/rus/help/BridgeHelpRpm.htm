Справка: Режим мост

На этой странице можно настроить порт LAN для работы в режиме "мост".

  • LAN4/ LAN3&LAN4 - Номер порта LAN для объединения в мост с портом WAN.

Примечание: Если изменить работу порта LAN в режим "мост" с портом WAN, система перезагрузится. Для сохранения настроек нажмите кнопку Сохранить.
