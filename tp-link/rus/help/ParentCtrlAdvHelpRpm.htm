Справка: Добавление/изменение записей родительского контроля

С помощью Маршрутизатора, который поддерживает функцию Родительского Контроля для контроля действий ребёнка в сети Интернет, Вы можете ограничить доступ ребёнка к определённым сайтам или задать время и расписание пользования сетью Интернет. На данной странице представлены инструкции по созданию подобных правил.

  • MAC-адрес контролируемого компьютера - Введите MAC-адрес компьютера, доступ которого в сеть Интернет Вы хотите контролировать. Вы также можете использовать функцию Все MAC-адреса в текущей локальной сети. Если данный раздел не будет заполнен, то созданное правило будет применено ко всем компьютерам, за исключением родительского.
  • Все MAC-адреса в текущей сети LAN - Вы можете просмотреть MAC-адреса всех компьютеров в данной локальной сети. Для этого необходимо нажать на кнопку выпадающего списка. Выберите один из представленных в списке; данный MAC-адрес будет добавлен к MAC-адресу компьютера ребёнка.
  • Описание сайта - В данном поле можно задать описание посещаемого сайта (сайтов). Следует помнить, что данное описание должно быть уникальным.
  • Имя разрешённого сайта - В данном поле Вы можете ввести до 8 доменных имён сайтов, посещать которые ребёнку разрешено. Можно ввести как целиком название, так и ключевые слова (например, google). Благодаря данному принципу доступ будет разрешён к любому сайту, доменное имя которого содержит указанное ключевое слово (www.google.com, news.google.com и т.д.).
  • Рабочее время - В данном поле Вы можете задать рабочее время для правила или с помощью меню Контроль доступа – Расписание создать собственное расписание в соответствии с Вашими предпочтениями. По умолчанию стоит Любое Время.
  • Состояние - В данном поле доступны два параметра настройки: Включить или Отключить. Включить означает, что правило работает, в то время, как Отключить означает, что правило не будет использоваться системой.

Нажмите Сохранить для завершения настройки.

Нажмите Назад для возврата на страницу Настроек Родительского Контроля.
