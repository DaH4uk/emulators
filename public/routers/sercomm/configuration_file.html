Конфигурация

Здесь Вы можете сохранить настройки роутера, восстановить ранее сохраненные настройки, и сбросить настройки маршрутизатора к заводским значениям.

Вы можете сохранить вашу конфигурацию в файл на компьютере или на USB диск.

Сохранить и восстановить настройки администратора

Сохранение на USB (предыдущее сохранение будет утеряно)
Сохранение на компьютере с защитой паролем
Восстановление из последней конфигурации, сохраненной на маршрутизаторе.
Восстановление конфигурации, сохраненной на компьютере.

Сохранить и восстановить конфигурации пользователя

Сохранение на USB (предыдущее сохранение будет утеряно)
Сохранение на компьютере с защитой паролем
Восстановление конфигурации из файла на USB
Восстановление конфигурации, сохраненной на компьютере.

Сброс к заводским настройкам

Нажатие кнопки Reset позволяет восстановить ваш маршрутизатор к заводским настройкам по умолчанию а также будет перезагрузит маршрутизатор
