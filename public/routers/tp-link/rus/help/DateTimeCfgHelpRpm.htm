Справка: Настройка времени

На этой странице вы сможете установить время вручную или настроить функцию автоматической синхронизации времени. В этом случае маршрутизатор сможет автоматически получать соответствующие данные от NTP-сервера через Интернет.

Часовой пояс - Выберите значение из выпадающего списка.

Чтобы установить время вручную:

  1. Выберите часовой пояс.
  2. Укажите Дату в формате месяц/день/год.
  3. Укажите Время в формате час/минута/секунда.
  4. Нажмите кнопку Сохранить.

Для автоматической синхронизации:

  1. Укажите адрес или доменное имя NTP-сервера 1 или NTP-сервера 2.
  2. Нажмите кнопку Получить среднее время по Гринвичу, чтобы получить среднее время по Гринвичу через Интернет.

Чтобы установить летнее время:

  1. Поставьте галочку в поле Включить летнее время, чтобы включить функцию летнего времени.
  2. Выберите правильные значения Начала и Конца летнего периода времени.
  3. Нажмите кнопку Сохранить.

Примечание:

  1. Данная настройка будет использоваться для ряда функций с временным критерием, например, для межсетевого экрана. Если значение времени не задано, эти функции не будут работать. Таким образом, является крайне важным указать эти значения сразу после входа в систему.
  2. Настройки времени будут утеряны при выключении маршрутизатора.
  3. При соответствующей настройке маршрутизатор будет получать среднее время по Гринвичу через Интернет в автоматическом режиме.
  4. В настройках летнего времени время начала не должно быть позднее времени окончания периода, который вы указываете для летнего времени.
