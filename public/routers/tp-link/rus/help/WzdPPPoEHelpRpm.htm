Справка: PPPoE/PPPoE Россия

Имя пользователя/Пароль - Укажите имя пользователя и пароль, предоставленные вашим поставщиком Интернет-услуг. Эти поля чувствительны к регистру.

Вторичное подключение - Вторичное подключение доступно только при использовании Интернет-подключения по протоколу PPPoE. Если ваш поставщик Интернет-услуг предоставляет вторичное подключение типа Динамический/Статический IP-адрес для подключения к локальной сети, выберите соответствующую кнопку для активации вторичного подключения по типу Динамический или Статический IP-адрес.

  • Отключено - По умолчанию вторичное подключение отключено, то есть используется только подключение по типу PPPoE/PPPoE Россия. Рекомендуется выбрать этот вариант.
  • Динамический IP-адрес - Использовать Динамический IP-адрес для подключения к локальной сети.
  • Статический IP-адрес - Использовать Статический IP-адрес для подключения к локальной сети. IP-адрес - Укажите здесь IP-адрес от вашего поставщика Интернет-услуг для вторичного подключения. Этот адрес используется только для доступа к локальной сети при вторичном подключении. Маска подсети - Укажите маску подсети от вашего поставщика Интернет-услуг для вторичного подключения.
