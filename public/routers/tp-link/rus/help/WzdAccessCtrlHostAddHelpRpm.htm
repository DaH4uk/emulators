Справка: Создать или изменить запись для узла

В этом окне вы можете создать запись для узла.

  • Имя узла - В этом поле необходимо ввести уникальное имя узла.
  • Режим - Здесь из выпадающего списка необходимо выбрать один из двух вариантов - "IP-адрес" или "MAC-адрес".

При выборе значения IP-адрес появится следующее поле:

  • IP-адрес в сети LAN - Укажите IP-адрес или диапазон адресов узла. Например, 192.168.0.23.

При выборе значения MAC-адрес появится следующее поле:

  • MAC-адрес - Укажите MAC-адрес узла в формате XX-XX-XX-XX-XX-XX. Например, 00-11-22-33-44-AA.

Нажмите кнопку Далее для продолжения, или Назад, чтобы вернуться на предыдущую страницу.
